# ShopEase

ShopEase is a premium, responsive e-commerce web application featuring curated standard products, an handloomed Indian **Saree Boutique**, and a custom **Gifts & Stationery** collection.

---

## 🚀 Key Features

* **Boutique & Gift Sections**: Custom luxury layout for heritage sarees and dedicated gifts collection.
* **User Authentication**: Client-side mock login and registration flows backed by storage persistence.
* **Persistent Cart & Wishlist**: Cart and liked products are saved securely in browser `localStorage`, persisting across page refreshes.
* **Theme System**: Smooth toggle between Light and Dark themes.

---

## 💻 Quick Start

Make sure you have [Node.js](https://nodejs.org) installed.

1. Navigate to the frontend directory:
   ```bash
   cd ecommerce-frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run local dev server:
   ```bash
   npm run dev
   ```
4. Build production bundle:
   ```bash
   npm run build
   ```
